# Vital User Guide Introduction

Vital is a "spectral warping wavetable synthesizer," meaning it generates sound using <InteractivePopover content={<Wavetable />}>wavetables</InteractivePopover> and offers advanced tools for manipulating their harmonic content. One of Vital's most distinctive features is its [Wavetable Editor](/docs/Vital/UserGuide/Wavetable-Editor), which provides a unique and powerful way to shape wavetables.

This guide aims to be a comprehensive resource for learning how to use Vital. It is divided into the following sections:

* **[Global Controls & Header Bar](/docs/Vital/UserGuide/Global-Controls-and-Header-Bar)**: Covers global keyboard shortcuts, the header bar, and the preset system.
* **[Modulation](/docs/Vital/UserGuide/Modulation)**: Covers Vital's modulation capabilities, including the Matrix tab.
* **[Oscillators and Sampler](/docs/Vital/UserGuide/Oscillators-and-Sampler)**: Covers the oscillators and sampler found in the Voice tab.
* **[Wavetable Editor](/docs/Vital/UserGuide/Wavetable-Editor)**: Covers the Wavetable Editor, which allows you to create and edit wavetables used by the oscillators.
* **[Filters](/docs/Vital/UserGuide/Filters)**: Covers the various filters available in the Voice tab.
* **[Envelopes & LFOs](/docs/Vital/UserGuide/Envelopes-and-LFOs)**: Covers the Envelopes and Low-Frequency Oscillators (LFOs) in the main UI.
* **[Effects](/docs/Vital/UserGuide/Effects)**: Covers the effects chain available in the Effects tab.
* **[Advanced](/docs/Vital/UserGuide/Advanced)**: Covers the Advanced tab which provides additional global and oscillator settings.
