\doxysubsection{Detailed Description}
A component that renders a full-\/screen overlay background using Open\+GL. 

The \doxylink{class_overlay_background_renderer}{Overlay\+Background\+Renderer} draws a simple rectangular overlay covering its bounds, using a specified color and optional additive blending. It\textquotesingle{}s designed to be used as the background layer for overlay components.
