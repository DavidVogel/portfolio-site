\doxysubsection{Detailed Description}
Displays a list of files (presets, wavetables, LFOs, samples) that can be selected and exported as a bank. 

The \doxylink{class_content_list}{Content\+List} class supports sorting by name, date, and selection status. It uses an Open\+GL-\/based rendering strategy for scrolling through potentially large lists of files efficiently.
