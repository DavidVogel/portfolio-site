\doxysubsection{Detailed Description}
A visual frame representing a single wavetable keyframe, which can be dragged by the user. 

The \doxylink{class_draggable_frame}{Draggable\+Frame} is used to indicate a wavetable keyframe\textquotesingle{}s position and allow mouse interaction, such as clicking, dragging, and selection. It can represent a full-\/frame keyframe or a smaller frame depending on the wavetable component type.
