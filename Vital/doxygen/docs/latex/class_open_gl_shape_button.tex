\doxysubsection{Detailed Description}
A Toggle\+Button that uses an \doxylink{class_open_gl_shape_button_component}{Open\+Gl\+Shape\+Button\+Component} for its rendering. 

This class acts as a JUCE Toggle\+Button but overrides its look with a custom Open\+GL shape.
