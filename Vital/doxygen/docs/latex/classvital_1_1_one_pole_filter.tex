A one-\/pole filter implementation with optional nonlinear saturation. 

This template class implements a one-\/pole filter with support for custom saturation functions. It can be used as a basic low-\/pass filter or as part of more complex filter structures, such as ladder or diode filters. By default, it does not apply saturation (using {\ttfamily \doxylink{namespacevital_1_1utils_a2a3a065cc4a3923cb0ac49a1e4231284}{utils\+::pass}}), but a saturation function can be provided to add nonlinear character to the filter.


\begin{DoxyTemplParams}{Template Parameters}
{\em saturate} & A function pointer to a saturation function that takes a \doxylink{structvital_1_1poly__float}{poly\+\_\+float} and returns a \doxylink{structvital_1_1poly__float}{poly\+\_\+float}. By default, it is {\ttfamily \doxylink{namespacevital_1_1utils_a2a3a065cc4a3923cb0ac49a1e4231284}{utils\+::pass}}, which applies no saturation. \\
