\doxysubsection{Detailed Description}
A UI component for exporting a selection of presets, wavetables, LFOs, and samples as a bank. 

The \doxylink{class_bank_exporter}{Bank\+Exporter} presents multiple \doxylink{class_content_list}{Content\+List} components for selecting items to include in a bank. Users can specify a bank name and export a ZIP file containing the selected items.
