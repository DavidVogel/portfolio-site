\doxysubsection{Detailed Description}
Represents a single state of a waveform at a specific position in a wavetable. 

\doxylink{class_wavetable_keyframe}{Wavetable\+Keyframe} acts as a snapshot of a waveform configuration at a particular frame index. Wavetable components typically have multiple keyframes spaced across the wavetable dimension. Interpolating between these keyframes produces smooth changes in the waveform over the course of the table.

Each keyframe stores\+:
\begin{DoxyItemize}
\item A position within the wavetable (0 to k\+Num\+Oscillator\+Wave\+Frames-\/1).
\item A reference to its owning component.
\end{DoxyItemize}

Subclasses define how the keyframeâ€™s waveform data is stored and how interpolation and rendering are performed.
