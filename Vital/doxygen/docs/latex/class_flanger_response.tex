\doxysubsection{Detailed Description}
Renders a visual representation of the flanger effect\textquotesingle{}s filter response. 

\doxylink{class_flanger_response}{Flanger\+Response} uses Open\+GL to draw the frequency response of a flanger effect. Users can interact with the response graph by clicking and dragging, which adjusts the associated sliders (center frequency and feedback) accordingly.
