\doxysubsection{Detailed Description}
A draggable magnification control for the envelope editor. 

The \doxylink{class_drag_magnifying_glass}{Drag\+Magnifying\+Glass} is an \doxylink{class_open_gl_shape_button}{Open\+Gl\+Shape\+Button} shaped like a magnifying glass. Dragging it allows the user to zoom in or out on the envelope editor area. Double-\/clicking resets the zoom.
