\doxysubsection{Detailed Description}
\subsubsection*{template$<$class Memory\+Type$>$\newline
class vital\+::\+Delay$<$ Memory\+Type $>$}
A flexible delay line effect processor that can operate in various styles and apply filtering. 

The \doxylink{classvital_1_1_delay}{Delay} class is templated on a Memory\+Type, allowing for different memory implementations (e.\+g., stereo or mono). It supports multiple delay styles (mono, stereo, ping-\/pong, damped, etc.) and can apply filtering to the delayed signal via internal one-\/pole filters.

Inputs\+:
\begin{DoxyItemize}
\item k\+Audio\+: The input audio signal.
\item k\+Wet\+: The wet mix percentage (0.\+0 to 1.\+0).
\item k\+Frequency\+: The base delay frequency in Hz (controls the delay time).
\item k\+Frequency\+Aux\+: The secondary delay frequency, used for stereo or ping-\/pong modes.
\item k\+Feedback\+: The amount of delayed signal fed back into the input.
\item k\+Damping\+: Controls the damping frequency applied in certain modes.
\item k\+Style\+: The style of delay (mono, stereo, ping-\/pong, etc.).
\item k\+Filter\+Cutoff\+: The cutoff frequency (in MIDI note) for internal filtering.
\item k\+Filter\+Spread\+: The frequency spread around the cutoff for the filter bands.
\end{DoxyItemize}

\doxylink{structvital_1_1_output}{Output}\+:
\begin{DoxyItemize}
\item One audio output containing the processed delay signal.
\end{DoxyItemize}

Styles\+:
\begin{DoxyItemize}
\item k\+Mono\+: A simple mono delay.
\item k\+Stereo\+: Stereo delay with separate delay times for left and right channels.
\item k\+Ping\+Pong\+: A ping-\/pong delay that alternates the delayed signal between left and right.
\item k\+Mid\+Ping\+Pong\+: A ping-\/pong style delay starting from a mid-\/image source.
\item k\+Clamped\+Dampened\+: \doxylink{classvital_1_1_delay}{Delay} with clamped dampening filters.
\item k\+Clamped\+Unfiltered\+: \doxylink{classvital_1_1_delay}{Delay} with clamped parameters but no filtering.
\item k\+Unclamped\+Unfiltered\+: Unclamped, no filtering delay (clean).
\end{DoxyItemize}

The delay time is derived from the input frequency parameters and can be modulated over time. Internal filtering is handled by one-\/pole high-\/pass and low-\/pass filters applied to the delayed signal.
