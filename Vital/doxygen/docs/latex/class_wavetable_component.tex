\doxysubsection{Detailed Description}
A base class representing a component in a wavetable synthesis chain. 

\doxylink{class_wavetable_component}{Wavetable\+Component} manages a collection of Wavetable\+Keyframes, each representing a waveform configuration at a certain position (time/frame index). By interpolating between these keyframes, the component produces a smooth transition of waveform shapes across the wavetable dimension. Wavetable\+Components can represent sources of waveforms, or modifiers that apply transformations to existing waveforms.

Key functionalities include\+:
\begin{DoxyItemize}
\item Creating and managing keyframes.
\item Interpolating between keyframes using linear or cubic styles.
\item Serialization to/from JSON for preset saving. 
\end{DoxyItemize}
