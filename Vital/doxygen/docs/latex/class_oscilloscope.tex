\doxysubsection{Detailed Description}
Renders a time-\/domain waveform using Open\+GL. 

The \doxylink{class_oscilloscope}{Oscilloscope} class displays a waveform in real-\/time by reading from a memory buffer of audio samples. It uses the \doxylink{class_open_gl_line_renderer}{Open\+Gl\+Line\+Renderer} base class to handle rendering a line representing the waveform.
