\doxysubsection{Detailed Description}
A UI component that displays the current playhead position in a wavetable editor. 

\doxylink{class_wavetable_playhead_info}{Wavetable\+Playhead\+Info} listens to a \doxylink{class_wavetable_playhead}{Wavetable\+Playhead} and shows the current frame position as a textual number. Whenever the playhead moves, this component updates and redraws the display.
