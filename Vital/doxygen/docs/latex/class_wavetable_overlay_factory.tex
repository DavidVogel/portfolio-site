\doxysubsection{Detailed Description}
A factory class for creating and configuring overlay components for wavetable editing. 

The \doxylink{class_wavetable_overlay_factory}{Wavetable\+Overlay\+Factory} is responsible for instantiating the appropriate overlay component based on the type of a \doxylink{class_wavetable_component}{Wavetable\+Component}. It also supports setting the overlay\textquotesingle{}s owner after creation to properly connect the overlay with the underlying wavetable component for editing.
