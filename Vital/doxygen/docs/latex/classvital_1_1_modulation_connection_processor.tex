\doxysubsection{Detailed Description}
A processor that applies a modulation signal to a parameter, performing mapping, scaling, and morphing. 

The \doxylink{classvital_1_1_modulation_connection_processor}{Modulation\+Connection\+Processor} takes a modulation input (e.\+g., from an LFO or envelope), applies transformations such as bipolar shifting, remapping via a \doxylink{class_line_generator}{Line\+Generator}, applying power curves, and scaling by a destination amount. It can handle both control-\/rate and audio-\/rate modulation, and supports polyphonic and stereo modulation. The output is a transformed modulation signal ready to be applied to a parameter in the synth.
