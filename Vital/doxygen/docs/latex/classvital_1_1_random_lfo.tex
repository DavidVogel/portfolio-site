\doxysubsection{Detailed Description}
A Low-\/\+Frequency Oscillator (LFO) that generates random modulation signals. 

The \doxylink{classvital_1_1_random_lfo}{Random\+Lfo} class creates a variety of random waveforms, including Perlin-\/like noise, sample-\/and-\/hold stepped values, sinusoidal interpolations, and chaotic Lorenz attractors. It supports resetting on triggers, stereo or mono modes, frequency control, and optional tempo synchronization via an external time reference.
