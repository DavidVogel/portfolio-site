\doxysubsection{Detailed Description}
Holds and manages a buffer of samples (\doxylink{structvital_1_1poly__float}{poly\+\_\+float}) for a \doxylink{classvital_1_1_processor}{Processor}\textquotesingle{}s output. 

This class also tracks trigger information such as trigger mask, value, and offset. It can run at audio or control rates, depending on buffer\+\_\+size.
