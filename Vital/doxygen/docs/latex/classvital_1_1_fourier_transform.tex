\doxysubsection{Detailed Description}
A Fourier transform implementation using Kiss\+FFT for platforms where other accelerations are unavailable. 

Kiss\+FFT is a simple, universally compatible \doxylink{classvital_1_1_f_f_t}{FFT} library. This implementation manually arranges real-\/only data into a complex form before performing forward and inverse transforms, and includes logic to manipulate the Nyquist component.

\begin{DoxyNote}{Note}
This class is used as a fallback when no other specialized \doxylink{classvital_1_1_f_f_t}{FFT} backends are available. 
\end{DoxyNote}
