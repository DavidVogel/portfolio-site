\doxysubsection{Detailed Description}
A specialized \doxylink{class_line_editor}{Line\+Editor} that visualizes and optionally animates a line-\/to-\/value mapping. 

The \doxylink{class_line_map_editor}{Line\+Map\+Editor} displays a mapping function or curve and can animate it in real-\/time based on an input signal (for instance, a parameter or modulation source). It inherits from \doxylink{class_line_editor}{Line\+Editor} and adds phase-\/based animation, allowing the visual curve to show a current position indicator that moves along the curve according to an input value.
