\doxysubsection{Detailed Description}
An overlay interface component for modifying a windowed section of a wavetable. 

\doxylink{class_wave_window_overlay}{Wave\+Window\+Overlay} provides controls to edit a "{}window"{} applied to a wavetable\textquotesingle{}s waveform. It allows setting left/right positions of the window and choosing a window shape. The overlay interacts with a \doxylink{class_wave_window_modifier}{Wave\+Window\+Modifier} and a \doxylink{class_wave_window_editor}{Wave\+Window\+Editor} to visually and interactively manipulate the window parameters for the currently selected keyframe.
