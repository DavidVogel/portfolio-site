\doxysubsection{Detailed Description}
Base class for all signal-\/processing units in Vital. 

A \doxylink{classvital_1_1_processor}{Processor} can have multiple inputs and outputs, manages its own state, and can be plugged into other Processors or have other Processors plugged in. In typical use, a {\ttfamily \doxylink{classvital_1_1_processor_router}{Processor\+Router}} manages connections and calls {\ttfamily \doxylink{classvital_1_1_processor_aa88622055da34fca35bb4192d524dd9d}{process()}}.
