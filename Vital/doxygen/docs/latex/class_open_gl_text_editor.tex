\doxysubsection{Detailed Description}
A text editor that uses an \doxylink{class_open_gl_image_component}{Open\+Gl\+Image\+Component} for rendering and updates the image on text changes. 

The \doxylink{class_open_gl_text_editor}{Open\+Gl\+Text\+Editor} applies a font and redraws its image on various text events, mouse events, and focus changes.
