\doxysubsection{Detailed Description}
A graphical editor for manipulating a single-\/cycle waveform\textquotesingle{}s sample values. 

The \doxylink{class_wave_source_editor}{Wave\+Source\+Editor} displays a waveform as a line renderer and supports editing the waveform by clicking and dragging. It can snap points to a user-\/defined grid and provides optional context menu actions like clearing or flipping the waveform. It also supports drag-\/and-\/drop of audio files that are translated into waveform data.
