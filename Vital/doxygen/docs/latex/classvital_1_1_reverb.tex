\doxysubsection{Detailed Description}
A \doxylink{classvital_1_1_processor}{Processor} implementing a dense feedback network reverb. 

The \doxylink{classvital_1_1_reverb}{Reverb} uses multiple all-\/pass filters and feedback delay lines to create a spacious, reverberant sound. It can be controlled via decay time, cutoff filters, chorus modulation, stereo width, and more.
