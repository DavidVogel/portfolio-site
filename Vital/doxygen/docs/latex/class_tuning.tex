\doxysubsection{Detailed Description}
A class for managing microtonal tunings and custom pitch mappings in Vital. 

The \doxylink{class_tuning}{Tuning} class allows loading and applying scales, .scl (Scala) files, .tun files, and keyboard mapping files (.kbm) to adjust how MIDI notes are mapped to frequencies. It supports setting a reference frequency, defining custom scales, and converting MIDI note numbers to pitch values according to the chosen tuning.
