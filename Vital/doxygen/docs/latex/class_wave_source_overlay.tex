\doxysubsection{Detailed Description}
Provides an overlay for editing \doxylink{class_wave_source}{Wave\+Source} objects in the Wavetable component. 

The \doxylink{class_wave_source_overlay}{Wave\+Source\+Overlay} is responsible for handling graphical user interface elements that allow users to edit a \doxylink{class_wave_source}{Wave\+Source} (time-\/domain waveform, frequency amplitudes, and phases). It integrates with various editors (\doxylink{class_wave_source_editor}{Wave\+Source\+Editor}, \doxylink{class_bar_editor}{Bar\+Editor}) to facilitate editing both the time-\/domain data and the frequency-\/domain data.
