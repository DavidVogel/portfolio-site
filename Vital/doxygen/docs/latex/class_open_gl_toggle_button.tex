\doxysubsection{Detailed Description}
A Toggle\+Button that uses an \doxylink{class_open_gl_button_component}{Open\+Gl\+Button\+Component} for its rendering. 

It supports various styles and uses Open\+GL for drawing, making it suitable for modern UIs.
