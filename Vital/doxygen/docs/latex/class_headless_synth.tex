\doxysubsection{Detailed Description}
A headless subclass of \doxylink{class_synth_base}{Synth\+Base} that provides a critical section and pausing capability. 

The \doxylink{class_headless_synth}{Headless\+Synth} can operate without a GUI, useful in environments like CLI tools, render farms, or automated test rigs. It uses a Critical\+Section to manage audio processing safely.
