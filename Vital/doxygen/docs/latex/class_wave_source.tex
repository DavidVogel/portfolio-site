\doxysubsection{Detailed Description}
A \doxylink{class_wavetable_component}{Wavetable\+Component} that acts as a direct source of waveforms. 

\doxylink{class_wave_source}{Wave\+Source} stores a series of Wave\+Source\+Keyframes, each holding a Wave\+Frame (time and frequency domain data). It supports two interpolation modes\+:
\begin{DoxyItemize}
\item Time interpolation\+: Interpolates directly between time-\/domain waveforms.
\item Frequency interpolation\+: Interpolates between frequency-\/domain representations, allowing smoother transitions with more control over harmonic phases and amplitudes.
\end{DoxyItemize}

This component provides a flexible basis for representing waves as raw data and morphing between them.
