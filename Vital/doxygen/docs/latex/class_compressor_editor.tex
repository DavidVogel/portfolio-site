\doxysubsection{Detailed Description}
A graphical interface component for editing a multiband compressor\textquotesingle{}s thresholds and ratios. 

The \doxylink{class_compressor_editor}{Compressor\+Editor} allows for the visualization and interactive editing of multiple compression bands (low, band, and high). It displays threshold lines, input/output levels, and ratio lines, and allows users to manipulate these values via mouse interaction. The component uses Open\+GL for efficient rendering.
