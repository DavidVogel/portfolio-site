\doxysubsection{Detailed Description}
Declares classes for time-\/domain memory storage and retrieval with cubic interpolation. 

The Memory and Stereo\+Memory classes store a history of samples in a ring buffer and allow retrieval of past samples using cubic interpolation. They are used to implement audio feedback loops, delays, or other time-\/domain manipulations efficiently.
