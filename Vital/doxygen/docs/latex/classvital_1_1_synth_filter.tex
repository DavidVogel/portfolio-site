\doxysubsection{Detailed Description}
Abstract base class for Vital’s synthesizer filters. 

\doxylink{classvital_1_1_synth_filter}{Synth\+Filter} defines the interface and shared utilities for filters in the Vital synthesizer. It manages the internal \doxylink{classvital_1_1_synth_filter_1_1_filter_state}{Filter\+State} and offers a static method for creating specific filter models (analog, ladder, comb, etc.).
