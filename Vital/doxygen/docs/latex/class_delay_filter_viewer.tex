\doxysubsection{Detailed Description}
A viewer that displays and allows interaction with the delay effect\textquotesingle{}s filter response. 

This class uses an Open\+GL line renderer to show the frequency response of the delay\textquotesingle{}s filter. It supports dragging to modify filter cutoff and spread values. Observers implementing the \doxylink{class_delay_filter_viewer_1_1_listener}{Listener} interface can respond to mouse drag delta movements.
