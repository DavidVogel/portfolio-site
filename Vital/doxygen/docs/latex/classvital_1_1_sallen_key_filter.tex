\doxysubsection{Detailed Description}
A Sallen-\/\+Key style filter capable of multiple modes (12dB, 24dB, dual modes) with nonlinear drive and resonance. 

The \doxylink{classvital_1_1_sallen_key_filter}{Sallen\+Key\+Filter} implements a classic Sallen-\/\+Key topology using one-\/pole filters combined in various configurations to produce low-\/pass, high-\/pass, and band-\/pass responses, as well as dual and notch-\/band configurations. It applies nonlinear saturations and allows tuning of drive, resonance, and relative blends of filter outputs to create a wide variety of timbres.

Inputs\+:
\begin{DoxyItemize}
\item k\+Audio\+: The input audio signal.
\end{DoxyItemize}

Outputs\+:
\begin{DoxyItemize}
\item A single filtered audio output.
\end{DoxyItemize}

Users can set the cutoff frequency (via MIDI note or frequency), resonance, and drive. The filter dynamically adjusts its internal states and blends low, band, and high outputs based on the mode selected. This results in flexible filtering behavior suitable for various synthesis tasks.
