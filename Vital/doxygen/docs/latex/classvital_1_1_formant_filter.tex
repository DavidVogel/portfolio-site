\doxysubsection{Detailed Description}
A multi-\/formant filter for vocal/voicing effects in the Vital synthesizer. 

\doxylink{classvital_1_1_formant_filter}{Formant\+Filter} manages multiple SVF (state variable filter) instances (via \doxylink{classvital_1_1_formant_manager}{Formant\+Manager}) tuned to specific formant characteristics. It allows different formant styles and interpolation of formant parameters based on user input (e.\+g., formant blending across a grid).
