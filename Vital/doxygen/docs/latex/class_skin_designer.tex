\doxysubsection{Detailed Description}
A Document\+Window that allows interactive editing of the \doxylink{class_skin}{Skin}. 

The \doxylink{class_skin_designer}{Skin\+Designer} provides UI elements to load, save, and edit the current skin colors and values. Primarily used for development and customization, it is not typically shown in a production environment.
