\doxysubsection{Detailed Description}
A slider-\/based UI component that displays selectable tabs. 

The \doxylink{class_tab_selector}{Tab\+Selector} uses the slider\textquotesingle{}s value to determine which tab is selected. It displays each tab name horizontally and highlights the currently selected tab. The component can be integrated with custom look-\/and-\/feels and supports Open\+GL image rendering for efficient drawing.
