\doxysubsection{Detailed Description}
A graphical editor for envelope shapes with interactive points and power curves. 

The \doxylink{class_envelope_editor}{Envelope\+Editor} class provides an interface to visualize and edit a complex envelope (ADHSR) using interactive markers. The user can adjust delay, attack, hold, decay, sustain, and release times, as well as the curvature (power) of the envelope segments. The envelope is rendered using Open\+GL and supports zooming and hovering over individual envelope points or power handles.
