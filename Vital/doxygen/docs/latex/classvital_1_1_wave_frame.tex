\doxysubsection{Detailed Description}
Represents a single frame of a wavetable, containing both time-\/domain and frequency-\/domain data. 

A \doxylink{classvital_1_1_wave_frame}{Wave\+Frame} holds a waveform in both time-\/domain and frequency-\/domain forms. It allows for operations such as normalization, clearing, transforming between time and frequency domains, and other utility functions to manipulate or analyze the waveform.
