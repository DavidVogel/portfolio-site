\doxysubsection{Detailed Description}
An interface for classes that listen to MIDI-\/driven parameter changes. 

Implement this listener if you need to respond to MIDI values changing synth parameters, pitch wheel movements, mod wheel updates, or preset changes triggered via MIDI.
