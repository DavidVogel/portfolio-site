\doxysubsection{Detailed Description}
A static utility class to access parameter details globally. 

\doxylink{classvital_1_1_parameters}{Parameters} provides static methods to\+:
\begin{DoxyItemize}
\item Access parameter details by name or index.
\item Check if a parameter exists.
\item Get parameter display names and ranges.
\end{DoxyItemize}

It uses a static \doxylink{classvital_1_1_value_details_lookup}{Value\+Details\+Lookup} (lookup\+\_\+) internally.
