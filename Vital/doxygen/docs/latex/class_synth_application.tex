\doxysubsection{Detailed Description}
The main application class for Vital\textquotesingle{}s standalone version. 

\doxylink{class_synth_application}{Synth\+Application} sets up the command-\/line interface, the main window (if not running headless), and handles tasks such as exporting wavetables, rendering audio to files, and showing help/version information.
