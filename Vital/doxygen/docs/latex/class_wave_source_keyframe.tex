\doxysubsection{Detailed Description}
A keyframe that holds a single Wave\+Frame and supports various interpolation methods. 

A \doxylink{class_wave_source_keyframe}{Wave\+Source\+Keyframe} stores a Wave\+Frame that can represent a full cycle of a waveform in both time and frequency domains. It offers functions for linear and cubic interpolation in either time or frequency domains, enabling smooth transitions between different waveforms stored in other keyframes.
