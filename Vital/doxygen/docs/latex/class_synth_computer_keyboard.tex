\doxysubsection{Detailed Description}
Provides a computer-\/keyboard-\/based MIDI input mechanism for the Vital standalone application. 

\doxylink{class_synth_computer_keyboard}{Synth\+Computer\+Keyboard} maps certain keys on a computer keyboard to MIDI note on/off events, allowing users to play notes without a hardware MIDI controller. It also supports shifting the keyboard layout up and down octaves via dedicated keys.

This class implements JUCE\textquotesingle{}s Key\+Listener interface to receive and respond to keyboard events.
