\doxysubsection{Detailed Description}
A structure representing a single modulation connection between a modulation source and a destination parameter. 

Each \doxylink{structvital_1_1_modulation_connection}{Modulation\+Connection} links a {\ttfamily source\+\_\+name} (e.\+g., an LFO or envelope) to a {\ttfamily destination\+\_\+name} (a synth parameter). It holds a \doxylink{classvital_1_1_modulation_connection_processor}{Modulation\+Connection\+Processor} to apply transformations like scaling or mapping curves.

The {\ttfamily is\+Modulation\+Source\+Default\+Bipolar} method checks if the source typically produces bipolar values (e.\+g., LFOs). Connections can be reset, and by default start disconnected until {\ttfamily reset\+Connection} is called.
