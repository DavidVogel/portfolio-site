\doxysubsection{Detailed Description}
A UI component that allows editing of a windowing function applied to a waveform. 

The \doxylink{class_wave_window_editor}{Wave\+Window\+Editor} displays a graphical window shape that can be adjusted by dragging handles on the left and right sides. Users can interact with these handles (left and right) to shift the window\textquotesingle{}s range, and the resulting window shape is updated in real-\/time. Different window shapes can be applied and listeners can be notified of changes.
