\doxysubsection{Detailed Description}
Manages a keyboard layout mapping for a computer keyboard used as a MIDI input device. 

\doxylink{classvital_1_1_string_layout}{String\+Layout} stores a layout of characters that map to notes, and keys to shift the octave up or down.
