\doxysubsection{Detailed Description}
A specialized \doxylink{class_synth_slider}{Synth\+Slider} that represents a single modulation amount control. 

\doxylink{class_modulation_amount_knob}{Modulation\+Amount\+Knob} provides popup menus for removing, bypassing, and changing modulation parameters (bipolar, stereo). It can also represent auxiliary connections (chained modulation paths).
