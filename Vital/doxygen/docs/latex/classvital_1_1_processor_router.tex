\doxysubsection{Detailed Description}
A specialized \doxylink{classvital_1_1_processor}{Processor} that manages a directed graph of Processors and ensures correct processing order. 

\doxylink{classvital_1_1_processor_router}{Processor\+Router} maintains a list of Processors and \doxylink{classvital_1_1_feedback}{Feedback} objects, and ensures that the audio/control graph is executed in the correct order each audio block. It manages detection of cycles, insertion of \doxylink{classvital_1_1_feedback}{Feedback} nodes, and propagation of sample rate and oversampling changes.

This class is central to Vital\textquotesingle{}s flexible and modular signal routing architecture.
