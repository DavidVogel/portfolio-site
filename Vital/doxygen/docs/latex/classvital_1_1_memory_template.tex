\doxysubsection{Detailed Description}
\subsubsection*{template$<$size\+\_\+t k\+Channels$>$\newline
class vital\+::\+Memory\+Template$<$ k\+Channels $>$}
A template for a memory buffer that stores time-\/domain samples for one or more channels. 

\doxylink{classvital_1_1_memory_template}{Memory\+Template} provides a ring buffer of samples. Samples can be pushed into the buffer and later retrieved based on a certain delay or offset. It supports clearing sections of the buffer and ensures that size is always a power-\/of-\/two for indexing efficiency.
