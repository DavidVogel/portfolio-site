\doxysubsection{Detailed Description}
A device selector component rendered using Open\+GL for improved performance. 

This class provides a device selector (for audio and MIDI devices) wrapped in an Open\+GL component. It updates its rendered image whenever resized, ensuring a smooth UI experience.
