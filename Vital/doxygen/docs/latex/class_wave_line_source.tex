\doxysubsection{Detailed Description}
A \doxylink{class_wavetable_component}{Wavetable\+Component} that generates waveforms from a series of line segments. 

\doxylink{class_wave_line_source}{Wave\+Line\+Source} uses a {\ttfamily \doxylink{class_line_generator}{Line\+Generator}} internally, which defines a waveform by connecting a set of points with lines (optionally smoothed). Each point has both a position and a power parameter that influence the shape. Interpolation between keyframes allows for dynamic changes in the line-\/based waveform shape across the wavetable dimension.

By controlling the number of points and their arrangement, one can create simple waves (e.\+g., linear ramps) or more complex curves. The {\ttfamily pull\+\_\+power\+\_\+} parameter affects how interpolation behaves, allowing for nonlinear transitions.
