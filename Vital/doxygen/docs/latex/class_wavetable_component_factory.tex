\doxysubsection{Detailed Description}
Factory class for creating and identifying different types of Wavetable\+Components. 

\doxylink{class_wavetable_component_factory}{Wavetable\+Component\+Factory} allows creation of source and modifier components by type enumerations or by name strings. It provides\+:
\begin{DoxyItemize}
\item A set of enumerated Component\+Types representing both wave sources and waveform modifiers.
\item Functions to create a component instance from a Component\+Type or a string name.
\item Methods to retrieve human-\/readable names of components.
\item Separations of components into source and modifier categories. 
\end{DoxyItemize}
