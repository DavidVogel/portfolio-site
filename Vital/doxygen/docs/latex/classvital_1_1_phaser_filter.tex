\doxysubsection{Detailed Description}
A multi-\/stage phaser filter for the Vital synthesizer. 

The \doxylink{classvital_1_1_phaser_filter}{Phaser\+Filter} class applies up to three sets of all-\/pass filter stages, each creating notches/peaks in the frequency response. Adjustable parameters include resonance, drive, and peak distribution. The filter can operate in either "{}clean"{} mode or a more distorted mode, changing how saturation is applied to the filter feedback and input signals.
