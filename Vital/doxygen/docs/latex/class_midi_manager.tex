\doxysubsection{Detailed Description}
The \doxylink{class_midi_manager}{Midi\+Manager} class handles all incoming MIDI messages and directs them to the synthesizer engine. 

This class interfaces with both the JUCE MIDI subsystem and the Vital {\ttfamily Sound\+Engine}. It can\+:
\begin{DoxyItemize}
\item Process note on/off, pitch bend, mod wheel, aftertouch, and other MIDI control messages.
\item Handle MPE (MIDI Polyphonic Expression) messages if enabled.
\item Support MIDI learn functionality, allowing MIDI controls to be dynamically mapped to synth parameters.
\item Relay MIDI-\/driven preset changes and gather MIDI input from external sources. 
\end{DoxyItemize}

A nested type defining a map from MIDI controls to a map of parameter names and their Value\+Details.
