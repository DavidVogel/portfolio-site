\doxysubsection{Detailed Description}
A specialized \doxylink{classvital_1_1_memory_template}{Memory\+Template} for two-\/channel (stereo) audio. 

\doxylink{classvital_1_1_stereo_memory}{Stereo\+Memory} stores two channels of audio samples and can retrieve past samples for both channels simultaneously using cubic interpolation.
