\doxysubsection{Detailed Description}
An interface class linking the Vital synthesizer backend (\doxylink{class_synth_base}{Synth\+Base}) with a GUI. 

\doxylink{class_synth_gui_interface}{Synth\+Gui\+Interface} provides methods to\+:
\begin{DoxyItemize}
\item Update GUI controls based on internal parameter changes.
\item Notify the GUI of changes to modulations and parameters.
\item Connect and disconnect modulations through GUI interactions.
\item Load and save presets, including invoking dialog boxes if applicable.
\end{DoxyItemize}

On some builds (HEADLESS mode), these functionalities may be stubbed out, meaning the GUI does nothing. When GUI is available, it typically integrates with a \doxylink{class_full_interface}{Full\+Interface} GUI component that displays and updates Vitalâ€™s parameters and states.
