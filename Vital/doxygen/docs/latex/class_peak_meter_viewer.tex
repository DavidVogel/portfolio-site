\doxysubsection{Detailed Description}
A visual component to display a peak meter using Open\+GL. 

The \doxylink{class_peak_meter_viewer}{Peak\+Meter\+Viewer} class shows a peak meter (such as levels of audio signals) in a graphical form. It uses two sets of vertices\+: one for the current peak and one for a peak memory reading. The meter is displayed as a vertical bar that moves based on the dB values of the signal.
