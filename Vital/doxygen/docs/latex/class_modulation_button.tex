\doxysubsection{Detailed Description}
A component representing a modulation source or connection button. 

The \doxylink{class_modulation_button}{Modulation\+Button} displays and manages a modulation source, allowing the user to connect it to various parameters, disconnect existing connections, and visualize modulation activity. It supports interaction modes like selecting, dragging out to map modulation, and invoking context menus to disconnect modulations.
