\doxysubsection{Detailed Description}
A UI component for visually displaying and interacting with an audio sample waveform. 

The \doxylink{class_sample_viewer}{Sample\+Viewer} uses Open\+GL rendering to show a waveform of a loaded sample. It can react to audio file drops, updates from the synthesizer engine, and provides animated line boosts that visualize sample playback position.
