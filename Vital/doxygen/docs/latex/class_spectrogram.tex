\doxysubsection{Detailed Description}
Renders a frequency-\/domain representation (spectrogram) using Open\+GL. 

The \doxylink{class_spectrogram}{Spectrogram} class uses a Fourier transform to display the frequency content of audio data over time. It reads from a Stereo\+Memory buffer, applies a window function, performs an FFT, and then visualizes the amplitude across frequencies as a line. Both left and right channels are represented, and various frequency and dB range parameters can be set.
