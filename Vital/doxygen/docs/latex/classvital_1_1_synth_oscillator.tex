\doxysubsection{Detailed Description}
A core oscillator processor that generates audio by reading wavetable data with various effects. 

This class handles unison, spectral morphing, distortion, FM/\+RM modulation, and more. It inherits from \doxylink{classvital_1_1_processor}{Processor}, allowing it to be part of Vital\textquotesingle{}s modular audio graph.
