\doxysubsection{Detailed Description}
A \doxylink{classvital_1_1_processor}{Processor} that applies various types of distortion to an audio signal. 

The \doxylink{classvital_1_1_distortion}{Distortion} class provides several distortion algorithms such as soft clipping, hard clipping, bitcrushing, waveshaping (folding), and downsampling. Users can specify the distortion type and drive amount, and the class will transform the audio accordingly.
