Detailed Description

A WavetableComponent that applies slew-rate limiting to a wave’s time-domain signal.

SlewLimitModifier imposes a maximum rate at which the waveform’s amplitude can change between samples. By limiting the upward and downward rate of change (slew), it can soften transients and produce smoother waveform transitions. Users can control separate upward and downward slew limits.
