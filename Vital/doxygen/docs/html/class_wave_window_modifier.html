Detailed Description

A WavetableComponent that applies a windowing function to a waveformâ€™s head and tail.

WaveWindowModifier uses a specified window shape (cosine, half-sine, linear, square, or "wiggle") to smoothly taper the waveformâ€™s amplitude near its start and end. By adjusting the windowâ€™s left and right positions, the user can control how much of the wave is affected and how abruptly it transitions at the boundaries.
