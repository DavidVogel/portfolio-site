Detailed Description

A classic transistor ladder-style filter for the Vital synthesizer.

The LadderFilter class simulates a multi-stage (four-pole) ladder filter with drive, resonance, and various output slopes (12 dB, 24 dB, etc.). It supports different styles and pass blends for flexible filter curves.
