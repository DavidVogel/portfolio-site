Detailed Description

An overlay section that displays application information, device settings, and UI scaling options.

The AboutSection shows the application's name, version, logo, and offers controls to:

  • Configure device settings (via an OpenGlDeviceSelector).
  • Toggle update checks.
  • Adjust GUI size scaling. Clicking outside the info rectangle hides this section.
