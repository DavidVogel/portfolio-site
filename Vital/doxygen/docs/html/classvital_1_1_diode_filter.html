Detailed Description

A diode ladder filter implementation for the Vital synthesizer.

This class inherits from Processor and SynthFilter, providing a nonlinear diode ladder filter stage with high-pass filtering options and drive/resonance controls. It uses a combination of one-pole filters and saturation functions to emulate the diode ladder behavior.
