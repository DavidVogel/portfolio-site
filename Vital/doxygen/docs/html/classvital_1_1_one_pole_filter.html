Detailed Description

template<poly_float(*)(poly_float) saturate = utils::pass<poly_float>>
class vital::OnePoleFilter< saturate >

A one-pole filter implementation with optional nonlinear saturation.

This template class implements a one-pole filter with support for custom saturation functions. It can be used as a basic low-pass filter or as part of more complex filter structures, such as ladder or diode filters. By default, it does not apply saturation (using utils::pass), but a saturation function can be provided to add nonlinear character to the filter.

Template Parameters
saturateA function pointer to a saturation function that takes a poly_float and returns a poly_float. By default, it is utils::pass, which applies no saturation.
