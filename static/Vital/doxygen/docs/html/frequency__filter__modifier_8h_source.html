1/*
2Summary:
3FrequencyFilterModifier is a wavetable component that modifies the harmonic content of a wave by applying frequency-domain filters like low-pass, band-pass, high-pass, or comb filtering. FrequencyFilterModifierKeyframe stores parameters (cutoff, shape) that define how frequencies are allowed or attenuated. By interpolating between keyframes and optionally normalizing the result, this component enables flexible and expressive frequency shaping of wavetables.
4 */
