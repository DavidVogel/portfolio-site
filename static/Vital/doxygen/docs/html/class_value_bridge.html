Detailed Description

A parameter bridge that connects a vital::Value to an AudioProcessorParameter, allowing the host to manipulate and automate internal values within the Vital synth engine.

This class extends JUCE's AudioProcessorParameter to expose internal Vital parameters as host-facing automation parameters. It handles conversions between normalized (0.0 to 1.0) parameter values and the internal engine's value ranges, as well as skewing/scaling functions (e.g., exponential, quadratic) that convert between user-facing displays and engine values.
