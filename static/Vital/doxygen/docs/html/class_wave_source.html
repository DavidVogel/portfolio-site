Detailed Description

A WavetableComponent that acts as a direct source of waveforms.

WaveSource stores a series of WaveSourceKeyframes, each holding a WaveFrame (time and frequency domain data). It supports two interpolation modes:

  • Time interpolation: Interpolates directly between time-domain waveforms.
  • Frequency interpolation: Interpolates between frequency-domain representations, allowing smoother transitions with more control over harmonic phases and amplitudes.

This component provides a flexible basis for representing waves as raw data and morphing between them.
