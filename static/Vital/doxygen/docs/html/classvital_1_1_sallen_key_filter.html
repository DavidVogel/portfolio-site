Detailed Description

A Sallen-Key style filter capable of multiple modes (12dB, 24dB, dual modes) with nonlinear drive and resonance.

The SallenKeyFilter implements a classic Sallen-Key topology using one-pole filters combined in various configurations to produce low-pass, high-pass, and band-pass responses, as well as dual and notch-band configurations. It applies nonlinear saturations and allows tuning of drive, resonance, and relative blends of filter outputs to create a wide variety of timbres.

Inputs:

  • kAudio: The input audio signal.

Outputs:

  • A single filtered audio output.

Users can set the cutoff frequency (via MIDI note or frequency), resonance, and drive. The filter dynamically adjusts its internal states and blends low, band, and high outputs based on the mode selected. This results in flexible filtering behavior suitable for various synthesis tasks.
