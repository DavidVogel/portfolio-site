Detailed Description

A UI component for interactively editing a phase value using a horizontal dragging gesture.

The PhaseEditor displays a series of vertical lines (tick marks) that represent phase divisions. Users can click and drag horizontally to adjust the phase, and listeners can be notified of changes. Each tick line's position reflects the current phase offset.
