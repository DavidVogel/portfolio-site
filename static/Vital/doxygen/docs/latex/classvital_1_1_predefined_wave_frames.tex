\doxysubsection{Detailed Description}
Holds a set of predefined \doxylink{classvital_1_1_wave_frame}{Wave\+Frame} shapes that can be used as basic building blocks. 

\doxylink{classvital_1_1_predefined_wave_frames}{Predefined\+Wave\+Frames} generates standard wave shapes (sin, square, saw, etc.) in a \doxylink{classvital_1_1_wave_frame}{Wave\+Frame} format that can be used for initializing or shaping other wavetables.
