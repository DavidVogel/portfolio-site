\doxysubsection{Detailed Description}
A keyframe class holding horizontal and vertical warp parameters at a given position. 

\doxylink{class_wave_warp_modifier_1_1_wave_warp_modifier_keyframe}{Wave\+Warp\+Modifier\+Keyframe} stores two float parameters, {\ttfamily horizontal\+\_\+power\+\_\+} and {\ttfamily vertical\+\_\+power\+\_\+}, which control how strongly the waveform is warped along the horizontal (time) and vertical (amplitude) axes. By interpolating between keyframes, we can smoothly transition from one warp setting to another.

Asymmetric flags allow treating the waveform differently in its positive/negative or left/right sections.
