\doxysubsection{Detailed Description}
A specialized \doxylink{class_open_gl_toggle_button}{Open\+Gl\+Toggle\+Button} with additional functionality for the Vital synth. 

This button supports MIDI learn operations and can display different text or behavior depending on whether it is toggled. It also notifies registered Button\+Listeners of changes.
