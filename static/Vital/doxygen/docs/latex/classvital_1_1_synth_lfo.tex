\doxysubsection{Detailed Description}
A versatile Low-\/\+Frequency Oscillator (LFO) for audio synthesis, supporting multiple sync modes and smoothing options. 

The \doxylink{classvital_1_1_synth_lfo}{Synth\+Lfo} class generates low-\/frequency modulation signals from a \doxylink{class_line_generator}{Line\+Generator} source. It supports\+:
\begin{DoxyItemize}
\item Various sync types (triggered, synced to host/tempo, envelope-\/like, looping)
\item Phase offset adjustments, stereo offsets, delay times, fade-\/in times, and smoothing.
\item Both control-\/rate and audio-\/rate processing, adapting seamlessly between them. 
\end{DoxyItemize}
