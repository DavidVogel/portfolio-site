\doxysubsection{Detailed Description}
A parameter bridge that connects a {\ttfamily \doxylink{classvital_1_1_value}{vital\+::\+Value}} to an Audio\+Processor\+Parameter, allowing the host to manipulate and automate internal values within the Vital synth engine. 

This class extends JUCE\textquotesingle{}s {\ttfamily Audio\+Processor\+Parameter} to expose internal Vital parameters as host-\/facing automation parameters. It handles conversions between normalized (0.\+0 to 1.\+0) parameter values and the internal engine\textquotesingle{}s value ranges, as well as skewing/scaling functions (e.\+g., exponential, quadratic) that convert between user-\/facing displays and engine values.
