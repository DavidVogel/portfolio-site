\doxysubsection{Detailed Description}
A top-\/level section for managing all modulation connections, amounts, and related UI components. 

The \doxylink{class_modulation_manager}{Modulation\+Manager} handles\+:
\begin{DoxyItemize}
\item All modulation connections (adding, removing, and modifying them).
\item Displaying modulation meters on sliders and knobs.
\item Showing and hiding \doxylink{class_modulation_amount_knob}{Modulation\+Amount\+Knob} controls for each modulation connection.
\item Handling dragging and dropping of modulations, including auxiliary (chained) modulations.
\item Managing popup and expansion boxes for complex modulation routings.
\end{DoxyItemize}

It integrates with the synthesizer model (via \doxylink{class_synth_gui_interface}{Synth\+Gui\+Interface}) to reflect and change modulation states.
