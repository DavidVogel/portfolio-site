\doxysubsection{Detailed Description}
A container managing a fixed number of Modulation\+Connections. 

\doxylink{classvital_1_1_modulation_connection_bank}{Modulation\+Connection\+Bank} pre-\/allocates a fixed number of connections (k\+Max\+Modulation\+Connections). It provides methods to create new connections by assigning a source and destination to an available slot.
