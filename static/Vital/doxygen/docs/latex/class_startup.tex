\doxysubsection{Detailed Description}
A utility class that handles initialization tasks and compatibility checks upon Vital’s startup. 

The \doxylink{class_startup}{Startup} class provides methods to perform essential checks and setup when Vital is launched. This includes\+:
\begin{DoxyItemize}
\item Verifying installation and upgrade status.
\item Loading configuration data such as MIDI mappings and user preferences.
\item Ensuring the computer supports the required CPU features. 
\end{DoxyItemize}
