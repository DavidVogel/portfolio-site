\doxysubsection{Detailed Description}
An interface for receiving parameter change notifications from the \doxylink{class_value_bridge}{Value\+Bridge}. 

Implementors of this interface can register with a \doxylink{class_value_bridge}{Value\+Bridge} to be notified whenever the parameter changes. This allows internal synth parameters to remain in sync with externally automated parameters.
