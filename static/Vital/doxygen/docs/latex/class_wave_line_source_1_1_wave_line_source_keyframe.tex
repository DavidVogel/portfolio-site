\doxysubsection{Detailed Description}
A keyframe class that represents a particular line-\/based waveform configuration at a given position. 

Each \doxylink{class_wave_line_source_1_1_wave_line_source_keyframe}{Wave\+Line\+Source\+Keyframe} holds a {\ttfamily \doxylink{class_line_generator}{Line\+Generator}} which stores multiple points defining the wave shape. Points can be added, removed, have their positions and power values adjusted, and the shape can be smoothed. Interpolating between two keyframes merges their line definitions in a power-\/scaled manner, controlled by {\ttfamily pull\+\_\+power\+\_\+}.
