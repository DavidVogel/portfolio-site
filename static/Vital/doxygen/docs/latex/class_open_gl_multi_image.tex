\doxysubsection{Detailed Description}
A component for rendering multiple image quads using Open\+GL. 

The \doxylink{class_open_gl_multi_image}{Open\+Gl\+Multi\+Image} class manages a set of quads, each displaying a portion (or the entirety) of a single image texture. It supports drawing a configurable number of these image quads at once, each positioned and sized independently. This enables rendering multiple image instances efficiently with a single texture.
