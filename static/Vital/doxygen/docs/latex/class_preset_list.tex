\doxysubsection{Detailed Description}
A UI component displaying a list of presets with sorting, filtering, and favorite management. 

The \doxylink{class_preset_list}{Preset\+List} shows presets in a table-\/like layout with columns for name, style, author, date, and a favorite star. Users can filter, sort, and rename presets, as well as toggle favorites and select presets.
