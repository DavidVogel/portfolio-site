\doxysubsection{Detailed Description}
A \doxylink{classvital_1_1_processor}{Processor} implementing a comb-\/based filter with multiple feedback styles. 

The \doxylink{classvital_1_1_comb_filter}{Comb\+Filter} supports comb, positive flange, and negative flange feedback variations, with options to blend low/high filter responses or spread band filters.
