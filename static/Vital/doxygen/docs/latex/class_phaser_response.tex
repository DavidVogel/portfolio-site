\doxysubsection{Detailed Description}
Visualizes the frequency response of the phaser effect. 

This class uses Open\+GL to render a phaser filter\textquotesingle{}s response curve in real-\/time. It supports user interaction through mouse dragging on the visual to change parameters such as cutoff and resonance.
