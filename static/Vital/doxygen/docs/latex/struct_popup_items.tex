\doxysubsection{Detailed Description}
A hierarchical structure of popup menu items for a selector component. 

Each \doxylink{struct_popup_items}{Popup\+Items} instance can represent a menu (with nested items) or a single menu item, identified by an ID, a name, and a selected state.
