\doxysubsection{Detailed Description}
A base overlay component for editing and interacting with a wavetable component\textquotesingle{}s parameters. 

The \doxylink{class_wavetable_component_overlay}{Wavetable\+Component\+Overlay} class provides a structured overlay UI for wavetable editing. It supports customizable sections and line dividers, along with titles and associated controls. Derived classes can add their own controls (such as sliders and buttons) and update their positions and visuals according to the wavetable keyframes selected.
