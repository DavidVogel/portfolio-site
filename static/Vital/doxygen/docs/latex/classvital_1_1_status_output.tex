\doxysubsection{Detailed Description}
A helper class to track the "{}status"{} of a particular \doxylink{structvital_1_1_output}{Output} as a \doxylink{structvital_1_1poly__float}{poly\+\_\+float} value. 

The \doxylink{classvital_1_1_status_output}{Status\+Output} class allows reading and clearing of a status value from a given \doxylink{structvital_1_1_output}{Output}. It can accumulate values for different voices (via update calls), and also recognizes a special "{}clear"{} value.
