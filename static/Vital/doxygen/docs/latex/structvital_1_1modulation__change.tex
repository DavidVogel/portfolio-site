\doxysubsection{Detailed Description}
A structure describing changes to the modulation routing in the engine. 

A \doxylink{structvital_1_1modulation__change}{modulation\+\_\+change} is used to communicate updates in modulation connections\+: connecting, disconnecting, or changing modulation parameters. It includes pointers to sources, destinations, and switches controlling mono/poly modulation. The engine uses this to apply changes atomically to the modulation network.
