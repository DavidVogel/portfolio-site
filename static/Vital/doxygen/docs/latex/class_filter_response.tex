\doxysubsection{Detailed Description}
Displays the frequency response of various filter models in Vital. 

\doxylink{class_filter_response}{Filter\+Response} visualizes how a chosen filter (analog, digital, ladder, comb, diode, formant, etc.) affects the audio spectrum. It allows real-\/time interaction by clicking and dragging to change filter cutoff and resonance or formant positions. The rendering is done via Open\+GL for efficiency.
