\doxysubsection{Detailed Description}
A class responsible for creating complete wavetables from groups of wavetable components. 

\doxylink{class_wavetable_creator}{Wavetable\+Creator} combines multiple Wavetable\+Groups (each containing sources and modifiers) to produce a final Wavetable object. It supports\+:
\begin{DoxyItemize}
\item Managing multiple groups of components (sources/modifiers).
\item Rendering the entire wavetable by blending the outputs of all groups.
\item Loading wavetables from audio files and applying different load styles (splicing, vocoding, pitched analysis).
\item Initializing predefined wave shapes or line-\/based generators.
\item Normalizing and DC-\/removing the final table if desired.
\end{DoxyItemize}

The creator can serialize its state to JSON and restore it, integrating with preset/loading systems.
