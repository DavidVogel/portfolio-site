\doxysubsection{Detailed Description}
Holds metadata about a single parameter (control) in the Vital synthesizer. 

The \doxylink{structvital_1_1_value_details}{Value\+Details} structure describes how a parameter should be scaled, displayed, and interpreted. This includes minimum/maximum values, default values, display names, units, and the scaling function to apply (linear, exponential, etc.). It also supports indexed parameters for stepping through discrete values (e.\+g., enumeration fields).
