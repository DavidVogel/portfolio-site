\doxysubsection{Detailed Description}
A helper class for handling drag-\/and-\/drop of audio files into a JUCE component. 

\doxylink{class_audio_file_drop_source}{Audio\+File\+Drop\+Source} integrates with JUCE\textquotesingle{}s File\+Drag\+And\+Drop\+Target to respond when a user drags and drops an audio file over a component. It\+:
\begin{DoxyItemize}
\item Registers basic audio formats with an Audio\+Format\+Manager.
\item Checks if a dropped file is a supported audio format.
\item Notifies registered listeners when an audio file has been dropped and loaded.
\end{DoxyItemize}

Subclasses must implement \doxylink{class_audio_file_drop_source_a1c51d8f1947c2b8a9e70532eb68d790a}{audio\+File\+Loaded(const File\& file)} to define custom loading behavior, and can add listeners that also receive audio file load notifications.
