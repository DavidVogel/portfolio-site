\doxysubsection{Detailed Description}
A specific \doxylink{class_wavetable_keyframe}{Wavetable\+Keyframe} that uses the File\+Source’s audio buffer. 

\doxylink{class_file_source_1_1_file_source_keyframe}{File\+Source\+Keyframe} represents one "{}snapshot"{} of the file-\/based wavetable. It provides methods to render the wave frame based on a chosen fade/interpolation style and phase style, and can interpolate between keyframes or compute normalization scales.
