\doxysubsection{Detailed Description}
Holds the state of the LFO for either control-\/rate or audio-\/rate processing. 

Each \doxylink{structvital_1_1_synth_lfo_1_1_lfo_state}{Lfo\+State} stores timing and smoothing variables to ensure consistent and correct LFO behavior after triggers, fades, and delays.
