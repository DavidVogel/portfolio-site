\doxysubsection{Detailed Description}
A component for visualizing and interacting with an audio waveform. 

The \doxylink{class_audio_file_viewer}{Audio\+File\+Viewer} displays an audio waveform and allows the user to adjust the window position and size used by a \doxylink{class_file_source}{File\+Source}. It supports file drops to load audio files directly.
