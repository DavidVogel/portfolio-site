\doxysubsection{Detailed Description}
A graphical user interface component representing a filter section in the synthesizer. 

The \doxylink{class_filter_section}{Filter\+Section} displays and manages a variety of filter parameters, including filter models, styles, cutoff, resonance, and other filter-\/related controls. It can show different filter types (analog, ladder, digital, formant, comb, etc.) and dynamically adapts its UI to the selected filter model and style.

This class also manages input routing controls (oscillator inputs, sample input, serial filtering input) when necessary, and integrates with the preset selector for choosing different filter models and styles.
