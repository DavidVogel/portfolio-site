\doxysubsection{Detailed Description}
Interface for objects that want to be notified of \doxylink{class_preset_selector}{Preset\+Selector} events. 

Classes implementing this interface can be registered to the \doxylink{class_preset_selector}{Preset\+Selector} to receive callbacks when navigation buttons are clicked or when text is clicked.
