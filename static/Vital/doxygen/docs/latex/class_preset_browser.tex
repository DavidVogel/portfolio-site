\doxysubsection{Detailed Description}
A UI for browsing, loading, and organizing presets. 

The \doxylink{class_preset_browser}{Preset\+Browser} combines a \doxylink{class_preset_list}{Preset\+List} (for preset files) and a \doxylink{class_selection_list}{Selection\+List} (for folders), and includes searching, filtering by style, displaying metadata (author, comments), and links to browse or purchase more presets.
