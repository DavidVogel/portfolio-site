\doxysubsection{Detailed Description}
An overlay UI component for editing \doxylink{class_file_source}{File\+Source} Wavetable components. 

The \doxylink{class_file_source_overlay}{File\+Source\+Overlay} allows configuring the start position, window size, fade style, and other parameters for a File\+Source-\/based keyframe in a Wavetable. It provides controls for loading a file, adjusting window parameters, normalization, fade style, and phase style.
