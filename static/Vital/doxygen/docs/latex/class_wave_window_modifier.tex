\doxysubsection{Detailed Description}
A \doxylink{class_wavetable_component}{Wavetable\+Component} that applies a windowing function to a waveform’s head and tail. 

\doxylink{class_wave_window_modifier}{Wave\+Window\+Modifier} uses a specified window shape (cosine, half-\/sine, linear, square, or "{}wiggle"{}) to smoothly taper the waveform’s amplitude near its start and end. By adjusting the window’s left and right positions, the user can control how much of the wave is affected and how abruptly it transitions at the boundaries.
