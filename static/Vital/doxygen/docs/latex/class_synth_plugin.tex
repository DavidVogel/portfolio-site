\doxysubsection{Detailed Description}
An Audio\+Processor implementation for a synthesizer plugin. 

\doxylink{class_synth_plugin}{Synth\+Plugin} integrates a \doxylink{class_synth_base}{Synth\+Base} for synthesis engine management and JUCE\textquotesingle{}s Audio\+Processor for plugin-\/host communication. It manages parameters, state loading/saving, and interaction with the GUI. \doxylink{class_synth_plugin}{Synth\+Plugin} also interacts with \doxylink{class_value_bridge}{Value\+Bridge} objects to synchronize parameter changes between the synth engine and the host.
