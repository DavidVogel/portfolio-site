\doxysubsection{Detailed Description}
Core class responsible for handling note events, oversampling, and the main effects chain. 

The main synthesis engine that handles voices, modulation, effects, and other top-\/level audio processing components for the Vital synthesizer.

The \doxylink{classvital_1_1_sound_engine}{Sound\+Engine} manages the top-\/level module architecture for Vital\textquotesingle{}s effect and modulation engines, connecting note handling, oversampling, effect chaining, and modulation routing. It derives from \doxylink{classvital_1_1_synth_module}{Synth\+Module} for the base module framework and \doxylink{classvital_1_1_note_handler}{Note\+Handler} for note-\/level logic.

The \doxylink{classvital_1_1_sound_engine}{Sound\+Engine} is responsible for managing voices, applying modulation, handling note on/off events, applying oversampling, and routing audio through the effects chain. It integrates with a \doxylink{classvital_1_1_synth_voice_handler}{Synth\+Voice\+Handler} to manage polyphony, and it uses a \doxylink{classvital_1_1_reorderable_effect_chain}{Reorderable\+Effect\+Chain} to process the final audio output.
