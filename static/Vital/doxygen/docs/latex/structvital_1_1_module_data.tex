\doxysubsection{Detailed Description}
Holds various data structures that define the internal state of a \doxylink{classvital_1_1_synth_module}{Synth\+Module}. 

This includes owned mono processors, submodules, and maps of controls, modulation sources, destinations, and status outputs. It is shared among various parts of the \doxylink{classvital_1_1_synth_module}{Synth\+Module}.
