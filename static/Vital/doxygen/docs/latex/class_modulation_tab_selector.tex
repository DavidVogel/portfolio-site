\doxysubsection{Detailed Description}
A section of the GUI providing multiple modulation sources as tabs. 

The \doxylink{class_modulation_tab_selector}{Modulation\+Tab\+Selector} displays a number of \doxylink{class_modulation_button}{Modulation\+Button} elements, each representing a modulation source. It can show or hide these sources based on how many are currently in use, and allows the user to select one modulation source at a time. Depending on configuration, these sources are displayed either vertically or horizontally, and their visibility adjusts dynamically based on connected modulations.
