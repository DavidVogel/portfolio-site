\doxysubsection{Detailed Description}
A classic transistor ladder-\/style filter for the Vital synthesizer. 

The \doxylink{classvital_1_1_ladder_filter}{Ladder\+Filter} class simulates a multi-\/stage (four-\/pole) ladder filter with drive, resonance, and various output slopes (12 dB, 24 dB, etc.). It supports different styles and pass blends for flexible filter curves.
