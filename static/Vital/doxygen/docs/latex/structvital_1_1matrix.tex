\doxysubsection{Detailed Description}
A structure representing a 4x1 matrix of \doxylink{structvital_1_1poly__float}{poly\+\_\+float} rows. 

This structure holds four \doxylink{structvital_1_1poly__float}{poly\+\_\+float} rows and provides utility functions for operations such as transposing and interpolating between matrix states. It is designed to work with \doxylink{structvital_1_1poly__float}{poly\+\_\+float}, a SIMD-\/friendly vector type used throughout Vital\textquotesingle{}s DSP code.

Each {\ttfamily rowN} is a \doxylink{structvital_1_1poly__float}{poly\+\_\+float} vector representing one row of the matrix. Although named "{}matrix,"{} it essentially acts as a 4-\/row structure that can be manipulated for certain vectorized math operations, including transposing these rows into columns and performing interpolations.
