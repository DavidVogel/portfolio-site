\doxysubsection{Detailed Description}
A processor that generates an envelope signal based on typical ADSR (Attack, Decay, Sustain, Release) parameters. 

The \doxylink{classvital_1_1_envelope}{Envelope} class controls the amplitude of a signal over time based on configurable segments (\doxylink{classvital_1_1_delay}{Delay}, Attack, Hold, Decay, Sustain, and Release). It inherits from \doxylink{classvital_1_1_processor}{Processor}, allowing it to be integrated into a larger signal processing chain. The envelope can be triggered and retriggered, transitioning through states that define its shape.
