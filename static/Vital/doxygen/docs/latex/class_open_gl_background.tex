\doxysubsection{Detailed Description}
A class that manages and renders a background image using Open\+GL. 

The \doxylink{class_open_gl_background}{Open\+Gl\+Background} class is responsible for loading, updating, and displaying a background image with Open\+GL. It provides thread-\/safe updating of the background image, manages vertex buffers, and sets up shader attributes for rendering the image.
