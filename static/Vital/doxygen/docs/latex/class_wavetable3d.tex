\doxysubsection{Detailed Description}
Open\+GL-\/based component for visualizing wavetables in different formats. 

The \doxylink{class_wavetable3d}{Wavetable3d} component can render\+:
\begin{DoxyItemize}
\item A 3D representation of multiple frames of a wavetable.
\item A 2D single-\/frame waveform view.
\item A frequency amplitude view of the wavetable.
\end{DoxyItemize}

It supports loading wavetables from JSON, copying/pasting, resynthesis, and text-\/to-\/wavetable conversions. It also integrates with parameter modulation for frame, spectral morph, and distortion adjustments.
