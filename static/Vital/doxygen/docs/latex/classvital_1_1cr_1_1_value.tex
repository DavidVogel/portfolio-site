\doxysubsection{Detailed Description}
A control-\/rate variant of the \doxylink{classvital_1_1cr_1_1_value}{Value} processor. 

This class functions similarly to the main \doxylink{classvital_1_1cr_1_1_value}{Value} processor but only updates a single sample at a time, suitable for control-\/rate signals (e.\+g., modulation parameters).
