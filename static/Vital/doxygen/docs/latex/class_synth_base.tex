\doxysubsection{Detailed Description}
A base class providing foundational functionality for the Vital synthesizerâ€™s engine, UI integration, modulation system, MIDI mapping, and preset management. 

The \doxylink{class_synth_base}{Synth\+Base} class\+:
\begin{DoxyItemize}
\item Integrates a Sound\+Engine for audio generation and modulation.
\item Manages MIDI input through a \doxylink{class_midi_manager}{Midi\+Manager}.
\item Maintains a set of controls (parameters) and handles parameter changes from GUI, MIDI, or code.
\item Supports loading/saving presets, wavetables, and other stateful elements.
\item Provides a queue-\/based system for thread-\/safe modulation and parameter changes.
\item Offers methods to render audio to disk, load tunings, and initiate default or user presets.
\end{DoxyItemize}

The class can be integrated with a GUI through a \doxylink{class_synth_gui_interface}{Synth\+Gui\+Interface} or used headlessly (see \doxylink{class_headless_synth}{Headless\+Synth}). Subclasses must implement certain virtual methods to integrate with an audio processing environment.
