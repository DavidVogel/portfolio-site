\doxysubsection{Detailed Description}
A scrollable, selectable popup list of items. 

\doxylink{class_popup_list}{Popup\+List} displays a vertical list of items that can be hovered, selected, and scrolled through. It supports a listener for when a new item is selected or double-\/clicked.
