\doxysubsection{Detailed Description}
An aggregate grouping that pairs multiple (parallel) voices with a shared \doxylink{classvital_1_1_processor}{Processor} instance. 

Vital uses parallel voices within an \doxylink{structvital_1_1_aggregate_voice}{Aggregate\+Voice} to handle SIMD lanes efficiently.
