\doxysubsection{Detailed Description}
Defines the Synth\+Module class which extends Processor\+Router to form a building block of the Vital synthesizer, encapsulating controls, modulation sources, and sub-\/modules. 

A Synth\+Module is a higher-\/level construct that groups together multiple Processors, Inputs, Outputs, and controls, handling both mono and polyphonic modulation chains. It provides interfaces for creating and retrieving modulation sources and destinations, managing parameter controls, and nesting submodules.
