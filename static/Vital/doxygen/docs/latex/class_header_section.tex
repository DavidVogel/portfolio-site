\doxysubsection{Detailed Description}
The topmost section of the synthesizer GUI, displaying the logo, tab selector, preset controls, oscilloscope, and more. 

The \doxylink{class_header_section}{Header\+Section} manages\+:
\begin{DoxyItemize}
\item A logo section (\doxylink{class_logo_section}{Logo\+Section})
\item A tab selector for switching between main sections (e.\+g., VOICE, EFFECTS, MATRIX, ADVANCED)
\item A preset selector for loading and saving patches
\item Volume and display elements like oscilloscope or spectrogram
\item Temporary tabs and overlays (e.\+g., preset browser, bank exporter)
\end{DoxyItemize}

Listeners can respond to changes such as tab selections, preset loading, or bank exports.
