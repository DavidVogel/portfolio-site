\doxysubsection{Detailed Description}
Represents a single playing note/voice, including voice-\/state and event handling. 

Voices can be grouped into \doxylink{structvital_1_1_aggregate_voice}{Aggregate\+Voice} sets, with each group sharing a \doxylink{classvital_1_1_processor}{Processor}. This class stores the note data and manages transitions between states (on/off/sustain).
