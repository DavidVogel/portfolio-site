\doxysubsection{Detailed Description}
A state-\/variable filter (SVF) implementation, supporting multiple filter types (12/24 dB, shelving, dual modes). 

The \doxylink{classvital_1_1_digital_svf}{Digital\+Svf} class provides a flexible filter design that can morph between low-\/pass, high-\/pass, band-\/pass, notch, peak, and specialized dual filter modes. It optionally supports a basic or advanced processing path, drive compensation, and user-\/defined resonance bounds.
