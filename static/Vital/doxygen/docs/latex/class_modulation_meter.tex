\doxysubsection{Detailed Description}
A visual component that displays the current modulation amount applied to a slider parameter. 

The \doxylink{class_modulation_meter}{Modulation\+Meter} shows how a parameter (represented by a \doxylink{class_synth_slider}{Synth\+Slider}) is modulated by one or more sources. It can display modulation in either a linear or rotary style, depending on the slider it is associated with. The meter updates dynamically based on the current modulation values (mono and poly outputs) and shows the modulated range of the parameter.
