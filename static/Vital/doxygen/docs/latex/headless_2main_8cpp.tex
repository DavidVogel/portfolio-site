The main entry point for the headless tool. 

Processes command-\/line arguments to optionally load a preset or wavetable and render it to an audio file.

Usage\+:
\begin{DoxyItemize}
\item Provide a file path to load as a preset/wavetable.
\item Use flags like -\/o/--output for output file, -\/l/--length for render length, -\/m/--midi for MIDI notes, -\/b/--bpm for BPM, and -\/i/--render-\/images to produce image frames.
\end{DoxyItemize}
