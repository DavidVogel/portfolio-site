\doxysubsection{Detailed Description}
A component for rendering multiple quads using Open\+GL, with customizable colors, rounding, and other effects. 

The \doxylink{class_open_gl_multi_quad}{Open\+Gl\+Multi\+Quad} class manages and draws a set of rectangular quads on the screen via Open\+GL. Each quad can be positioned, sized, and given unique shader values. Various parameters like thickness, rounding, and alpha blending can be adjusted. This makes it suitable for complex GUI elements such as scroll bars, progress bars, or other shapes.
