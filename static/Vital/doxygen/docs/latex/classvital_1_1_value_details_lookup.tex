\doxysubsection{Detailed Description}
Maintains a lookup table for all parameters defined in Vital. 

\doxylink{classvital_1_1_value_details_lookup}{Value\+Details\+Lookup} stores and indexes all parameter metadata (\doxylink{structvital_1_1_value_details}{Value\+Details}) used by the synthesizer. It supports\+:
\begin{DoxyItemize}
\item Retrieving parameter details by name or index.
\item Checking if a given name is a parameter.
\item Adding parameter groups (like envelopes, LFOs, etc.) with prefixed names and IDs.
\item Accessing parameter ranges and display names. 
\end{DoxyItemize}
