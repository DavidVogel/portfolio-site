\doxysubsection{Detailed Description}
A utility class for loading and saving configuration, preset, and state data for the Vital synthesizer. 

This class provides a comprehensive set of static methods for\+:
\begin{DoxyItemize}
\item Converting between data formats (e.\+g., float PCM data and integer PCM).
\item Reading and writing synth states, including parameters, wavetables, samples, LFOs, modulations, and more.
\item Managing user directories, favorites, installed packs, and additional folders.
\item Updating old configuration/preset states to newer versions.
\item Handling checks for updates, authentication, window size, and other global settings. 
\end{DoxyItemize}
