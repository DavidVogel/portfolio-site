\doxysubsection{Detailed Description}
A \doxylink{classvital_1_1_synth_module}{Synth\+Module} and \doxylink{classvital_1_1_note_handler}{Note\+Handler} that manages a pool of polyphonic voices, handles note-\/on/off logic, and routes the data to multiple processors. 

The \doxylink{classvital_1_1_voice_handler}{Voice\+Handler} class takes in note events (MIDI or otherwise) and dispatches them to available or stolen voices, tracks voice states (e.\+g., sustaining), and provides outputs that other modules can use to render audio or handle modulation based on active voices.
