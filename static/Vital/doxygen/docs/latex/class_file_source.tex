\doxysubsection{Detailed Description}
A \doxylink{class_wavetable_component}{Wavetable\+Component} that uses an external audio sample as its source. 

\doxylink{class_file_source}{File\+Source} loads a segment of audio data (e.\+g., from a sample file) and transforms it into wavetables. It provides multiple interpolation (fade) styles and phase manipulation options for constructing a wavetable frame. Users can set how the audio frames are extracted (window size, fade style), and how phases are handled (e.\+g., vocoding).
