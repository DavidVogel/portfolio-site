\doxysubsection{Detailed Description}
A keyframe class holding parameters for phase modification at a given position. 

\doxylink{class_phase_modifier_1_1_phase_modifier_keyframe}{Phase\+Modifier\+Keyframe} stores a phase offset and a mix amount. It can apply various phase modification styles to a Wave\+Frameâ€™s frequency domain. Interpolation between keyframes allows animating phase changes over a wavetable.
