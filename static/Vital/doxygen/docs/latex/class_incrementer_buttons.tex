\doxysubsection{Detailed Description}
A pair of buttons for incrementing and decrementing a Slider\textquotesingle{}s value. 

The \doxylink{class_incrementer_buttons}{Incrementer\+Buttons} class provides two small triangular arrow buttons, one for incrementing and one for decrementing the value of an associated Slider. When either button is pressed, the Slider\textquotesingle{}s value is adjusted by one unit. The appearance of the buttons adapts to the currently active skin colors, and can be toggled active or inactive.
