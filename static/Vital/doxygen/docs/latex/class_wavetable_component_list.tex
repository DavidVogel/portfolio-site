\doxysubsection{Detailed Description}
A UI component that lists and manages the wavetable sources and modifiers. 

This class displays a scrollable list of wavetable source groups and their modifiers. It allows adding, removing, and reordering sources and modifiers, as well as resetting them. It interacts with \doxylink{class_wavetable_creator}{Wavetable\+Creator} to maintain and modify the structure of the wavetable.
